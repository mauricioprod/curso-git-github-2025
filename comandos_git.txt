git init . 		-> Cria um repo git na pasta atual
git status 		-> Mostra o status do trabalho
git add nome_arquivo 	-> Adiciona o arquivo para ser commitado
git commit -m "msg" 	-> Faz commit do que estava para ser comitado
git lot 		-> Mostra todo o histórico de commit


## Fluxo de trabalho
1. criar diretório do projeto
2. iniciar git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome do arquivo>
5. git status
7. git commit -m "mensagem"
8. git status
9. volta ao passo 3
